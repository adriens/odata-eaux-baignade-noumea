# ‚ö° For impatients

Just run the following script in any shell and enjoy:

```shell
#!/bin/sh
# Diplay latest flags
sh <(curl https://tea.xyz) +duckdb.org \
duckdb << EOF
INSTALL httpfs;
LOAD httpfs;
select plage,
    flag_color,
    case 
        when (flag_color = 'BLUE')      THEN 'üü¶'
        when (flag_color = 'YELLOW')    THEN 'üü®'
        when (flag_color = 'RED')       THEN 'üü•'
    end as flag_color
from read_csv_auto('https://bit.ly/3ZCJ1X5') as latest;
EOF

```

### last-20

```shell
#!/bin/sh
# Diplay latest flags
sh <(curl https://tea.xyz) +duckdb.org \
duckdb << EOF
INSTALL httpfs;
LOAD httpfs;
select date, plage,
    case 
        when (flag_color = 'BLUE')      THEN 'üü¶'
        when (flag_color = 'YELLOW')    THEN 'üü®'
        when (flag_color = 'RED')       THEN 'üü•'
    end as flag
from
read_csv_auto('https://bit.ly/3mAUIPr')
order by date desc
limit 10;
EOF
